# UI Standard

**Preferred framework:** PySide6

## Requirements

- Dark theme compatible
- Dockable when DCC supports docking
- No blocking UI
- Separate UI from logic

## UI must follow

- QWidget based
- No global variables
- Signals over direct calls

## Style

- Background: `#1b1b1b`
- Rounded buttons
- Minimal text
- Icon-first design

👉 Keep consistency with MonoAnimTools.
