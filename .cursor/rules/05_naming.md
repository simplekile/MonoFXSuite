# Naming Convention

**Language:** English only

| Element | Convention |
|---------|------------|
| Folders | `snake_case` |
| Classes | `PascalCase` |
| Functions | `snake_case` |
| Variables | `snake_case` |

## UI & Controllers

- UI classes must end with: **UI**
- Controllers must end with: **Controller**

### Example

```
ribbon_trail/
  RibbonTrailUI
  RibbonTrailController
```
